Detailed Description

template<typename StateType>
class bfb::StateMachine< StateType >

StateMachine implements a finite state machine. A standby state should be provided. A StateMachine object can be limited to a certain type of state, via templating. If something else needs to see the state of the machine, it should probably be a part of the machine.
