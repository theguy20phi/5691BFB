Detailed Description

Issue provides a basis for managing and logging problems that may occur during driving. Issues should be constructed as static and constant, in the event the problem occurs. Issues are logged in vector that can be globally read from, but only modified by Issue objects. Issues are sorted by highest to lowest severity when logged.
