Detailed Description

template<typename StateType>
class bfb::StateMachine< StateType >

StateMachine implements a queue-based finite state machine, similar to a stack-based state machine. States are enqueued, and when a state completes its behaviour, dequeued. A standby state should be provided. A StateMachine object can be limited to a certain type of state, via templating. If something else needs to see the state of the machine, it should probably be a part of the machine.
