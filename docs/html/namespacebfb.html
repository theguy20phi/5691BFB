Detailed Description

A simple interface for controllers.

Author
Braden Pierce

Pidf implements a basic PID+ff controller, with additions to prevent integral windup and derivative kick, as well as determine if the controller is settled.

Author
Braden Pierce

Tbh implements a Take-Back-Half controller, essentially a modified integrator, with the addition of a method to check if the controller is settled.

Author
Braden Pierce

The mathUtil file implements math related methods and functions that are of use in many files.

Braden Pierce
