Detailed Description

Wrapper class for out-the-box pros Imu class to simplify the interface to essentially a yaw gyroscope (I'm not concerned with acceleration nor other axii), change "heading" to a more common axis (Z/Yaw) + remove wrap, add in a filter behind the scenes, and combine information from multiple IMUs.
